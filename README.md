# 个人轻量级云盘与媒体中心 (Personal Cloud & Media Center)

本项目提供了一套完整的自动化脚本集合，用于在 Linux 服务器（如 VPS）上快速搭建基于 S3 对象存储的个人云盘、离线下载节点以及流媒体播放中心。

## 架构概览

* **底层存储**：使用 JuiceFS 将 S3（如 Wasabi, Tebi）挂载为本地目录。
* **离线下载**：使用 Aria2 进行全协议（HTTP/BT/Magnet）离线下载。
* **文件管理**：使用 CHFS (Cute Http File Server) 提供网页端文件管理和分享。
* **媒体串流**：提供轻量级的 Python Web 服务，用于网页相册浏览和支持断点续传/拖动的视频推流。

## 🚀 快速开始与全量使用说明

建议使用 **root** 用户执行以下操作，以避免权限问题。

### 1. 挂载云存储 (JuiceFS)
**脚本**: `01_juice.sh`

此脚本用于安装 JuiceFS 客户端，并将远程 S3 存储桶挂载到服务器的 `/mnt/` 目录下。

* **运行方法**：
  bash 01_juice.sh
* **选项说明**：
  * 选择 `1` (新建)：输入你的 S3 存储桶名称、Bucket URL、AK 和 SK。
  * 选择 `2` (恢复)：如果服务器重启或挂载掉线，使用此选项可以一键扫描并恢复所有挂载点。
* **⚠️ 核心提示 (元数据数据库位置)**：
  JuiceFS 的本地 SQLite 元数据文件（`.db` 结尾的隐藏文件）**默认保存在 `/root/` 目录下**（例如 `/root/.wasabi.db`）。请妥善保管这些文件，如果丢失，将无法读取云端数据！

### 2. 部署离线下载引擎 (Aria2)
**脚本**: `02_aria2.sh`

此脚本用于一键安装和配置 Aria2 后台下载服务。

* **运行方法**：
  bash 02_aria2.sh
* **使用流程**：
  1. 运行脚本后选择 `1` 安装 Aria2。
  2. 安装完成后，脚本会输出配置信息（包含 RPC 端口、RPC 密钥和下载目录）。
  3. 建议通过脚本选项 `7` 将下载目录修改为挂载好的 JuiceFS 目录（例如 `/mnt/downloads`），这样下载的文件会直接写入云盘。
* **💻 图形化控制端 (AriaNg) 用法**：
  Aria2 本身没有图形界面，需要搭配 Web GUI 使用。推荐使用 AriaNg All-In-One 版本：
  1. 下载 GUI 客户端压缩包：[AriaNg-1.3.13-AllInOne.zip](https://github.com/mayswind/AriaNg/releases/download/1.3.13/AriaNg-1.3.13-AllInOne.zip)
  2. 在你的个人电脑上解压该 ZIP 文件。
  3. 双击打开解压出来的 `index.html` 文件（浏览器中打开）。
  4. 点击左侧的“AriaNg 设置” -> “RPC (xxxx:6800)”，填入你服务器的 IP 地址、Aria2 的 RPC 端口以及刚刚生成的 RPC 密钥，刷新页面提示“已连接”即可开始添加下载任务。

### 3. 启动网页文件浏览器 (CHFS)
**脚本**: `03_chfsma.sh`

用于启动 CHFS，提供一个可视化的网页网盘界面，方便上传、下载和管理文件。

* **运行方法**：
  bash 03_chfsma.sh
* **访问方式**：
  默认端口为 `81`。在浏览器中访问 `http://你的服务器IP:81` 即可。
  *(注：默认配置在 `chfs/chfs.ini` 中，映射目录为 `/mnt/`，具体账号密码规则请参考该 ini 文件)*

### 4. 媒体流与娱乐服务
**脚本**: `05_phoo.py` / `06_vidd.py` / `06.py`

这些是轻量级的 Python 媒体服务，按需启动。

* **网页图库 (`05_phoo.py`)**：
  python3 05_phoo.py
  运行在 `8090` 端口。访问 `http://你的服务器IP:8090`，可以以瀑布流/网格形式优雅地浏览 `/mnt` 目录下的图片。

* **视频推流服务 (`06_vidd.py` 推荐 / `06.py`)**：
  python3 06_vidd.py
  运行在 `8091` 端口。这是一个支持 HTTP 206 断点续传的流媒体服务器，专门优化了视频拖动体验（秒跳转）。运行后，在 VLC 播放器或浏览器中输入 `http://你的服务器IP:8091` 即可播放选定目录内的视频；`06.py` 还额外支持生成 `.m3u` 播放列表用于连播。

### 5. 安全卸载与数据备份
**脚本**: `99_unjuice.sh`

当你需要重启服务器、迁移数据或备份元数据时，**强烈建议**先运行此脚本。

* **运行方法**：
  bash 99_unjuice.sh
* **作用**：
  1. 安全弹出所有 JuiceFS 挂载点。
  2. **强制合并 SQLite WAL 日志**：将 `/root/` 目录下的 `.db-wal` 日志数据合并写入到主 `.db` 文件中。只有合并后，你直接复制 `/root/.*.db` 文件进行备份才是完整和安全的。

## ⚠️ 防火墙放行端口列表
请确保你的云服务器安全组或系统防火墙（如 `ufw`, `iptables`）放行了以下端口，否则外部无法访问：
* `6800` (Aria2 RPC 默认端口)
* `81` (CHFS 文件服务器)
* `8090` (图片画廊服务)
* `8091` (视频流媒体服务)
